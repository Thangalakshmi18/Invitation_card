# Wedding Invitation Site

A personal online invitation site for weddings and special events.

## Features

- Responsive design that adapts to mobile, tablet, and desktop
- Beautiful background images for visual appeal
- Wedding details display with staggered typing animations
- RSVP link integration
- Elegant styling with romantic colors and fonts

## How to Use

1. Open `index.html` in your web browser
2. Customize the content in the HTML file
3. Update the RSVP link in `script.js`
4. Replace background images in the `../assets/` folder
5. Modify styles in `styles.css` as needed

## Customization

- **Content**: Edit the text and wedding details in `index.html`
- **Background Images**: Update the image paths in `styles.css` (currently using `../assets/` from parent folder)
- **Styling**: Modify `styles.css` for colors, fonts, and layout
- **Functionality**: Update `script.js` for interactive features
- **RSVP Link**: Change the URL in the RSVP link setup

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, and background images)
- Vanilla JavaScript

## Browser Support

Works in all modern browsers that support CSS Grid and Flexbox.
